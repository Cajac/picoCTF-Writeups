### Get the flag

Next, we check for included files and the changes made on them with `git log`
```bash

└─$ cd drop-in   

┌──(kali㉿kali)-[/mnt/…/picoCTF_2024/General_Skills/Blame_Game/drop-in]
└─$ ls -la

drwxrwxrwx 1 root root    0 Jun 10 10:20 ..

┌──(kali㉿kali)-[/mnt/…/picoCTF_2024/General_Skills/Blame_Game/drop-in]
```
